# Dota Pro Tracker Parser

CLI-инструмент для анализа статистики героев Dota 2 из [Pro Tracker](https://dota2protracker.com).

## Возможности

- Получение статистики героев по позициям (1-5)
- Фильтрация по минимальному количеству матчей (500+)
- Сортировка по винрейту
- Кликабельные ссылки в терминале для быстрого перехода на страницу героя

## Установка

```bash
# Клонируйте репозиторий
git clone <your-repo-url>
cd dota-pro-tracker-parser

# Установите зависимости (не требуется для собранной версии)
npm install

# (Опционально) Установите глобально
npm link
```

## Сборка

Проект использует [Bun](https://bun.sh) для сборки:

```bash
# Собрать в один минифицированный файл
bun run build

# Собрать standalone исполняемый файл (не требует Node.js)
bun run build:exe

# После сборки можно запустить
./dist/dota-tracker -pos 1
```

## Использование

### Локальный запуск

```bash
# Через npm
npm start -- -pos 1

# Напрямую через node
node index.js -pos 1
```

### После глобальной установки

```bash
dota-tracker -pos 1
```

### Параметры

- `-pos <1-5>` - позиция героя (1 = carry, 2 = mid, 3 = offlane, 4 = soft support, 5 = hard support)

### Примеры

```bash
# Лучшие кэрри (позиция 1)
dota-tracker -pos 1

# Лучшие мидеры (позиция 2)
dota-tracker -pos 2

# Лучшие оффлейнеры (позиция 3)
dota-tracker -pos 3

# Лучшие саппорты (позиция 4-5)
dota-tracker -pos 4
dota-tracker -pos 5
```

## Результат

Команда выводит список героев, отсортированных по винрейту:

```
Abaddon - 55.83%
Juggernaut - 54.10%
Lifestealer - 52.96%
...
```

Имена героев являются кликабельными ссылками (работает в современных терминалах).
